Formulas to Know Cold

  • Slope = (y₂ - y₁) / (x₂ - x₁) — rise over run
  • a/b = c/d → ad = bc — cross multiply
  • P(event) = favorable / total — probability
  • P(not A) = 1 - P(A) — complement
  • d = √[(x₂-x₁)² + (y₂-y₁)²] — distance
  • mid = ((x₁+x₂)/2, (y₁+y₂)/2) — midpoint
  • y = mx + b — slope-intercept
  • a² - b² = (a+b)(a-b) — difference of squares
  • % = (part / whole) × 100 — percentages
